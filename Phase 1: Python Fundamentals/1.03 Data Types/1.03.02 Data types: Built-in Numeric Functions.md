# Built-in Numeric Functions in Python

1. abs()
	•	Returns the absolute value of a number.
	•	Example:
		```
		python
		print(abs(-10)) # 10
		print(abs(10)) # 10
		print(abs(-10.5)) # 10.5
		```

2. round()
	•	refer to 1.03.01 Data types: numeric


3. sum()
	•	Returns the sum of all items in an iterable.
    •	iterable: list, tuple, set, etc.
	•	Example:
		```
		python
		print(sum([1, 2, 3, 4, 5])) # 15
		print(sum([1, 2, 3, 4, 5], 10)) # 25, here 10 is the starting value, so 10 + 1 + 2 + 3 + 4 + 5 = 25
		```
    •	syntax: sum(iterable, start=0)


4. min()
	•	Returns the smallest item in an iterable.
	•	Example:
		```
		python
		print(min([1, 2, 3, 4, 5])) # 1
		print(min([1, 2, 3, 4, 5], 10)) # 1
		```
    •	syntax: min(iterable, start=0)


5. max()
	•	Returns the largest item in an iterable.
	•	Example:
		```
		python
		print(max([1, 2, 3, 4, 5])) # 5
		print(max([1, 2, 3, 4, 5], 10)) # 10
		```
    •	syntax: max(iterable, start=0)


6. pow()
	•	Returns the power of a number.
	•	Example:
		```
		python
		print(pow(2, 3)) # 8
		print(pow(2, 3, 5)) # 3, here 5 is the modulus, so (2^3) % 5 = 8 % 5 = 3
		```
    •	syntax: pow(base, exp, mod)


7. divmod()
	•	Returns the quotient and remainder of a division.
	•	Example:
		```
		python
		print(divmod(10, 3)) # (3, 1), here 10 // 3 = 3 and 10 % 3 = 1
		```
    •	syntax: divmod(a, b)


8. is_integer()
	•	Returns True if a float is an integer, False otherwise.
	•	Example:
		```
		python
		print((10.0).is_integer()) # True
		print((10.5).is_integer()) # False
		```
    •	syntax: float.is_integer()
