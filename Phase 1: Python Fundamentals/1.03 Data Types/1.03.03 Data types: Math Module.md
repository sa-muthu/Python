# Math Module in Python
	The math module is a built-in module in Python that provides access to mathematical functions.
    The below are some math module functions:

! Note:
    Module is a file containing Python code that defines functions, classes, variables, or runnable code, which you can reuse in other Python programs.
    To use a module, you need to import it.
    Example:
    ```
    python
    import math # at the top of the file
    print(math.pi) # wherever you want to use it
    ```
    syntax: module_name.function_name (module name is math in this case)


1. math.pi
	•	Returns the value of pi.
	•	Example:
		```
		python
		import math
		print(math.pi)
		```


2. math.e (Euler's number)
	•	Returns the value of e.
	•	Example:
		```
		python
		import math
		print(math.e)
		```


3. math.ceil(x)
	•	Returns the smallest integer greater than or equal to x.
	•	Example:
		```
		python
		import math
		print(math.ceil(10.5)) # 11
        print(math.ceil(-10.5)) # -10
        print(math.ceil(10.00001)) # 11
        print(math.ceil(10.99999)) # 11
        print(math.ceil(10)) # 10
		```


4. math.floor(x)
	•	Returns the largest integer less than or equal to x.
	•	Example:
		```
		python
		import math
		print(math.floor(10.5)) # 10
        print(math.floor(-10.5)) # -11
        print(math.floor(10.00001)) # 10
        print(math.floor(10.99999)) # 10
        print(math.floor(10)) # 10
		```


5. math.trunc(x)
	•	Returns the truncated value of x (the integer part of x).
	•	Example:
		```
		python
		import math
		print(math.trunc(10.5)) # 10
        print(math.trunc(-10.5)) # -10
        print(math.trunc(10.00001)) # 10
        print(math.trunc(10.99999)) # 10
        print(math.trunc(10)) # 10
		``` 



6. math.fabs(x) # alternative to abs()
	•	Returns the absolute value of x.
	•	Example:
		```
		python
		import math
		print(math.fabs(-10)) # 10.0
		print(math.fabs(10)) # 10.0
		print(math.fabs(-10.5)) # 10.5
		```


7. math.sqrt(x)
	•	Returns the square root of x.
	•	Example:
		```
		python
		import math
		print(math.sqrt(16)) # 4.0
		print(math.sqrt(16.5)) # 4.062019202317536
		print(math.sqrt(-16)) # ValueError: math domain error
		```
	
        
### More math functions can be found in the official documentation (https://docs.python.org/3/library/math.html) or by typing help(math) in the python interpreter
