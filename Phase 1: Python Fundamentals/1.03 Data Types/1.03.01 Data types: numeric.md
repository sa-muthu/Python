# Numbers in Python

1. Number Types
	•	Integer (int): Whole numbers, positive or negative, without a decimal component.
            •	Arbitrary precision
			•	Example: 10, -5, 0, 1000
    
    •	Float (float): Real numbers with a decimal component.
            •	Limited precision (usually 64-bit floating-point)
			•	Example: 10.5, -5.2, 0.0, 1000.5
    
    •	Complex (complex): Numbers with a real and imaginary part.
            •	Example: 10 + 5j, -5.2 + 2.1j, 0 + 0j
			

2. Arithmatic Operators
	•	Addition (+)
    	•	Example: 10 + 5 = 15
	•	Subtraction (-)
    	•	Example: 10 - 5 = 5
	•	Multiplication (*)
    	•	Example: 10 * 5 = 50
	•	Division (/)
    	•	Example: 10 / 5 = 2.0
	•	Floor Division (//)
    	•	Example: 10 // 5 = 2
	•	Modulo (remainder) (%)
    	•	Example: 10 % 5 = 0
	•	Exponentiation (**)


3. Operator Precedence and Parentheses
	•	Parentheses () - Highest precedence
	•	Exponentiation (**) - Second highest precedence
	•	Multiplication (*) and Division (/) - Third highest precedence
	•	Addition (+) and Subtraction (-) - Lowest precedence
	•	Example:
		```
		python 
		x = 10 + 5 * 2
		print(x) # 20
		```
		

4. Type Conversion and Type Checking
	•	Use the type() function to check the type of a variable.
	•	Example:
		```
		python 
		print(type(10)) # <class 'int'>
		print(type(10.5)) # <class 'float'>
		print(type(10 + 5j)) # <class 'complex'>
		```

    •	Use the int(), float(), and complex() functions to convert between types.
	•	Example:
		```
		python 
		print(int(10.5)) # 10
		print(float(10)) # 10.0
		print(complex(10, 5)) # (10+5j)
		```


5. Type Promotion
	•	Type promotion is the process of converting a number from one type to another.
	•	Example:
		```
		python 
		print(10 + 5.5) # 15.5 (int + float = float)
		print(10 + 5j) # (10+5j) (int + complex = complex)
		print(10.5 + 5j) # (10.5+5j) (float + complex = complex)
		```

6. Integer Division vs. Float Division
	•	Integer division (//) returns the whole number part of the division.
	•	Float division (/) returns the exact result of the division.
	•	Example:
		```
		python 
		print(10 // 3) # 3
		print(10 / 3) # 3.3333333333333335
		```
	
7. Rounding
	•	Use the round() function to round a number to the nearest integer.
	•	Example:
		```
		python 
        print(round(10.5)) # 10
        print(round(10.6)) # 11
        print(round(10.5, 1)) # 10.5
        print(round(10.55, 1)) # 10.5
        print(round(10.555, 2)) # 10.56
        ```
