1. Variable
	•	A variable is a name or label that refers to a value stored in memory.
	•	Think of it like a box with a name on it.
	•	Example:
	```
	python 
	x = 10
	```
	Here, x is the variable. It’s a name that points to the value 10.


2. Variable Names
	•	Variable names are identifiers that refer to values stored in memory.
	•	Example:
	```
	python 
	name = "Alice"
	```
	•	Here, name is the variable. It’s a name that points to the value "Alice".


3. Variable Values
	•	A variable value is the data that is stored in a variable.
	•	Example:
	```
	python 
	name = "Alice"
	```
	•	Here, "Alice" is the value of the variable name.


4. Dynamic Typing
	•	Python is a dynamically typed language, which means you don’t need to declare the type of a variable when you create it.
	•	Example:
	```
	python 
	x = 10  # Integer
	y = 20.5  # Float
	z = "Hello"  # String
	```
	•	Python automatically determines the type of the variable based on the value assigned to it.	


5. Variable Names Rules
	•	Variable names must start with a letter (a-z, A-Z) or an underscore (_).
	•	They cannot start with a number.
	•	They can only contain letters, numbers, and underscores.
	•	They are case-sensitive (x and X are different).
	•	They cannot be Python keywords (like if, else, for, etc.).
	•	Example:
	```
	python 
	valid_var = 10
	_another_var = 20
	2invalid_var = 30  # Starts with a number
	```

6. Checking Varible Type
	•	Use the type() function to check the type of a variable.
	•	Example:
	```
	python 
	print(type(x)) # <class 'int'>
	print(type(name)) # <class 'str'>
	```


7. Memory Identity (or Memory Address)
	•	Memory identity is the unique identifier of a variable.
	•	Use the id() function to get the memory identity of a variable.
	•	Example:
	```
	python 
	x = 10
	print(id(x)) # 140735191166480 (example Id)
	```
	

8. Deleting Variables
	•	Use the del keyword to delete a variable.
	•	Example:
	```
	python 
	del x
	```
	•	After deleting a variable, it cannot be used.
		```
		python 
		print(x) # NameError: name 'x' is not defined
		```
	

9. Assignment
	•	Assignment is the process/action of giving a value to a variable.
	•	In Python, we use the equals sign (=) for assignment.
	•	Example:
	```
	python 
	x = 10
	name = 'Delilah'
	```
	
	•	The first line assigns 10 to the variable x.
	•	The second line assigns 'Alice' to the variable name.


10. Reassignment
	•	Reassignment is the process of changing the value of a variable.
	•	Example:
	```
	python 
	x = 10
	x = 20  # Reassigned
	```
	•	The value of x is now 20. It was 10 before.


11. Multiple Assignment
	•	You can assign multiple values to multiple variables in a single line.
	•	Example:
	```
	python 
	x, y, z = 10, 20, 30
	```
	•	Here, x gets 10, y gets 20, and z gets 30.



12. Augmented Assignment Operators
	•	Augmented assignment operators are shorthand operators that combine an arithmetic operation with an assignment operation.
	•	Example:
	```
	python 
	x += 5  # Equivalent to x = x + 5
	x -= 5  # Equivalent to x = x - 5
	x *= 5  # Equivalent to x = x * 5
	x /= 5  # Equivalent to x = x / 5
	x %= 5  # Equivalent to x = x % 5
	x **= 5  # Equivalent to x = x ** 5 (power)
	```
