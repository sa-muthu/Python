# 1.02.02 Variables and Assignments Exercises

Practice makes perfect! Work through these exercises to reinforce your understanding of Python variables and assignment operators.

---

### Exercise 1: Variable Creation and Printing
Create three variables with the following names and values:
- `first_name`: Your first name (string)
- `last_name`: Your last name (string)
- `age`: Your favorite number (integer)

Then, use a single `print()` statement to display them on one line.

### Exercise 2: Naming Rules
Identify which of the following variable names are **invalid** in Python and explain why:
1. `my_variable_1`
2. `1_variable`
3. `my-variable`
4. `_secret`
5. `class`
6. `Variable2`

### Exercise 3: Dynamic Typing
1. Create a variable called `data` and assign it the value `10.5`.
2. Print the type of `data` using the `type()` function.
3. Reassign `data` to the string `"Python"`.
4. Print the type of `data` again.

### Exercise 4: Multiple Assignment
In a single line of code, assign the value `5` to variables `a`, `b`, and `c`. Then, print the sum of all three variables.

### Exercise 5: Augmented Assignment
Start with a variable `score = 100`. Perform the following operations using **augmented assignment operators** and print the result after each step:
1. Add `50` to the score.
2. Subtract `20` from the current score.
3. Multiply the current score by `2`.
4. Divide the current score by `4`.

### Exercise 6: Memory Identity (id)
1. Create two variables `x = 500` and `y = 500`.
2. Print the `id()` of both `x` and `y`.
3. Are they the same? (Note: Python sometimes optimizes memory for small integers, but larger ones usually have different IDs).
4. Assign `y = x`. Print the `id()` again. Are they the same now?

### Challenge: Swapping Variables
Given two variables:
```python
a = "Up"
b = "Down"
```
Swap their values so that `a` becomes `"Down"` and `b` becomes `"Up"`. 
**Constraint:** Try to do this in only **one** line of code using multiple assignment!
