# Aggregation in Pandas 📊

Aggregation is the process of computing a single summary value (like a mean, sum, or count) from a group of data points. This is essential for transforming raw data into high-level insights.

---

## 1. Global Summary Statistics

Pandas can quickly summarize every numerical column in your entire dataset. This is perfect for a "first look" at your data.

```python
import pandas as pd

# Load the dataset
# Ensure Pokemon.csv is in your working directory
df = pd.read_csv("Pokemon.csv")

# Get the average of ALL numerical columns
# Note: numeric_only=True ignores text columns to avoid errors
print("--- Average Values ---")
print(df.mean(numeric_only=True))

# The "Ultimate Overview"
# Returns count, mean, std, min, percentiles, and max
print("\n--- Detailed Summary ---")
print(df.describe())
```

---

## 2. Aggregating Specific Columns

Often, you only care about one or two specific metrics. You can call aggregation methods directly on a Series (column).

### Numerical Metrics
```python
avg_height = df['Height'].mean()
max_weight = df['Weight'].max()

print(f"Average Pokémon Height: {avg_height:.2f}m")
print(f"Heaviest Pokémon Weight: {max_weight}kg")
```

### Categorical Metrics
For non-numerical data, we usually aggregate by counting frequencies or finding unique entries.

```python
# How many of each Primary Type exist?
type_counts = df["Type1"].value_counts()
print(type_counts.head()) # Show top 5 types

# List all unique types in the dataset
unique_types = df["Type1"].unique()
print(f"\nUnique Types found: {len(unique_types)}")
```

---

## 3. Targeted Aggregation

You can combine **Filtering** with **Aggregation** to answer specific questions.

> **Question:** What is the average weight of Legendary Pokémon?

```python
# 1. Filter for Legendaries
legendary_mask = df["Legendary"] == 1
legendary_df = df[legendary_mask]

# 2. Aggregate the weight
avg_legendary_weight = legendary_df["Weight"].mean()

print(f"Average Legendary Weight: {avg_legendary_weight:.2f}kg")
```

---

## 4. Powerful Grouping with `.groupby()`

The `groupby` method is the gold standard for aggregation. It splits the data into groups (e.g., by Type) and applies a function to each group.

```python
# Group by 'Type1' and find the average height for each type
type_groups = df.groupby("Type1")

# Extract the 'Height' and get the mean
avg_height_by_type = type_groups["Height"].mean()

# Sort results to see the tallest types on top
print(avg_height_by_type.sort_values(ascending=False))
```

---

## 5. Multiple Aggregations at Once

With `.agg()`, you can compute multiple statistics simultaneously.

```python
# For each type, get the Min, Max, and Mean of Weight
weight_stats = df.groupby("Type1")["Weight"].agg(['min', 'max', 'mean'])
print(weight_stats.head())
```
