# Pandas Series

Pandas series is a one-dimensional labeled array that can hold any data type. It is synonymous with a column in a spreadsheet.

## 1. Converting a Data into a Pandas Series

For normal conversion of data to a series, you can use the function `pd.Series(input)`.

### Example: Converting a List
```python
import pandas as pd

data = [100, "a", "b", 400.34, True] # this is the data
series = pd.Series(data) # conversion of data to series

# NOTE: The 'S' in Series is Capitalized
print(series)
```

### Custom Indexing
By default, data is stored with index starting from 0. To use custom indices, use the `index` keyword argument.

```python
series_with_index = pd.Series(data, index=["Integer", "Vowel", "Consonent", "Float", "Boolean"])
print(series_with_index)
```

## 2. Accessing Data Inside the Series

You can access data using labels or integer positions:

- **`.loc["index_name"]`**: Locate by index label.
- **`.iloc[index]`**: Locate by integer position.

You can also use these methods to modify data using the assignment operator.

### Examples:
```python
# Accessing
print(series_with_index.loc["Consonent"])
print(series_with_index.iloc[2])

# Modifying
series_with_index.loc["Consonent"] = "G"
series_with_index.iloc[1] = "E"
print(series_with_index)
```

## 3. Filtering By Value

You can filter values by certain conditions using the syntax `name[condition]`.

### Example:
```python
numerical_data = [1, 5, 3, 4, 3, 2, 5, 5, 6, 3, 4, 5, 7, 3, 4, 2, 3]
numerical_data_series = pd.Series(numerical_data)

larger_datas = numerical_data_series[numerical_data_series > 4]
print(larger_datas)
```

## 4. Mathematical Operations

Pandas provides built-in methods to perform mathematical operations on a series.

### Examples:
```python
print(numerical_data_series.sum())            # Sum of values
print(numerical_data_series.mean())           # Average
print(numerical_data_series.median())         # Middle value
print(numerical_data_series.mode())           # Most frequent value
print(numerical_data_series.std())            # Standard deviation
print(numerical_data_series.var())            # Variance
print(numerical_data_series.min())            # Minimum value
print(numerical_data_series.max())            # Maximum value
print(numerical_data_series.count())          # Count of non-NA values
print(numerical_data_series.unique())         # Unique values
print(numerical_data_series.value_counts())   # Count of each unique value
print(numerical_data_series.describe())       # Description of the series (you can use this to see all the above functions)
print(help(pd.Series))                        # You dont have to remember all this, you can use this to see all the above
```

## 5. Creating a Series from a Dictionary

You can directly convert a dictionary into a series where keys become indices.

```python
data = {"a": 1, "b": 2, "c": 3}
data_series = pd.Series(data)
print(data_series)
```

## 6. Creating a Series from a NumPy Array

```python
import numpy as np
data = np.array([1, 2, 3, 4, 5])
data_series = pd.Series(data)
print(data_series)
```

## 7. Creating a Series from a Scalar Value

If you provide a single value and an index, Pandas will repeat the value for each index.

```python
data = 10
data_series = pd.Series(data, index=["a", "b", "c", "d", "e"])
print(data_series)
```
