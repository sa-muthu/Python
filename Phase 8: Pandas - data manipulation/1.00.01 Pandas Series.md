# Pandas Series 

A Pandas **Series** is a one-dimensional labeled array. You can think of it as a single column in an Excel spreadsheet or a more powerful version of a Python list.

---

## 1. Creating a Series

You can create a Series from lists, dictionaries, or even scalars using `pd.Series()`.

### From a List
```python
import pandas as pd

# Heterogeneous data is supported
data = [100, "Python", 400.34, True]
series = pd.Series(data)

# Note: Indices default to 0, 1, 2...
print(series)
```

### Custom Indexing
Labels make your data much easier to work with.

```python
labels = ["ID", "Topic", "Score", "Active"]
series_labeled = pd.Series(data, index=labels)

print(series_labeled)
```

---

## 2. Accessing & Modifying Data

Pandas provides two primary ways to locate data:
- **`.loc[]`**: Access by **Label** (the custom index).
- **`.iloc[]`**: Access by **Integer Position** (0, 1, 2...).

```python
# Accessing
print(f"Topic: {series_labeled.loc['Topic']}")
print(f"Index 1: {series_labeled.iloc[1]}")

# Modifying
series_labeled.loc['Score'] = 450.0
print(series_labeled['Score'])
```

---

## 3. Mathematical Operations

If your Series contains numerical data, Pandas offers powerful built-in statistics.

```python
scores = pd.Series([85, 90, 78, 92, 88])

print(f"Average: {scores.mean()}")
print(f"Max:     {scores.max()}")
print(f"Count:   {scores.count()}")

# Get everything at once!
print(scores.describe())
```

---

## 4. Vectorized Filtering

You can filter a Series using simple boolean conditions. No `for` loops required!

```python
# Keep only scores above 85
high_scores = scores[scores > 85]
print(high_scores)
```

---

## 5. Other Data Sources

### From a Dictionary
Keys become the index automatically.
```python
dict_data = {"Mon": 10, "Tue": 20, "Wed": 30}
series_from_dict = pd.Series(dict_data)
```

### From a Scalar
Repeat a single value across multiple indices.
```python
constant_series = pd.Series(5, index=["a", "b", "c"])
```
