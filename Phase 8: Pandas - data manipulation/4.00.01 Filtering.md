# Filtering in Pandas

Filtering is the process of keeping only the rows that match specific conditions. This is essential for focusing your analysis on relevant data subsets.

## 1. Filtering by Value

You can use standard comparison operators (`>`, `<`, `>=`, `<=`, `==`, `!=`) to filter rows based on numerical or categorical values.

```python
import pandas as pd

data = pd.read_csv("Pokemon.csv")

# Rows where Height is 5 or more
print(data[data["Height"] >= 5])

# Rows where Weight is 300 or more
print(data[data["Weight"] >= 300])
```

---

## 2. Filtering by Multiple Conditions

Combine multiple conditions using bitwise operators:
- **`&` (AND)**: Both conditions must be true.
- **`|` (OR)**: At least one condition must be true.

> [!IMPORTANT]
> When using multiple conditions, you **must** wrap each condition in parentheses `()`.

```python
# Height >= 5 AND Weight >= 300
print(data[(data["Height"] >= 5) & (data["Weight"] >= 300)])

# Height >= 5 OR Weight >= 300
print(data[(data["Height"] >= 5) | (data["Weight"] >= 300)])

# Combining AND and OR
print(data[(data["Height"] >= 5) & (data["Weight"] >= 300) | (data["Legendary"] == 1)])
```

---

## 3. Filtering by List (`.isin()`)

Use `.isin()` to filter rows where a column's value matches any item in a provided list.

```python
# Select Pokémon with Type1 as Fire, Water, or Grass
print(data[data["Type1"].isin(["Fire", "Water", "Grass"])])

# Multiple conditions with isin()
print(data[data["Type1"].isin(["Fire", "Water", "Grass"]) & (data["Legendary"] == 1)])
```

---

## 4. Filtering by String (`.str.contains()`)

To search for a substring within a column, use `.str.contains()`.

```python
# Select Pokémon whose name contains "Mega"
print(data[data["Name"].str.contains("Mega")])
```

---

## 5. Filtering Null and Not Null Values

Identify missing data using `.isnull()` and non-missing data using `.notnull()`.

```python
# Rows where Type2 is missing
print(data[data["Type2"].isnull()])

# Rows where Type2 is NOT missing
print(data[data["Type2"].notnull()])
```

---

## 6. Filtering by Index (`.iloc[]`)

Use `.iloc[]` to filter by integer-based positions.

```python
# First 10 rows
print(data.iloc[0:10])

# First 10 rows and first 3 columns
print(data.iloc[0:10, 0:3])
```
