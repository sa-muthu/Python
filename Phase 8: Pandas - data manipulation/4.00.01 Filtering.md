# Filtering in Pandas 

Filtering allows you to zoom in on the data that meets specific criteria. It uses **Boolean Indexing** to create a mask and apply it to the DataFrame.

---

## 1. Basic Filtering

The syntax is `df[condition]`. The condition usually involves a comparison operator (`>`, `<`, `==`, etc.).

```python
import pandas as pd

df = pd.read_csv("Pokemon.csv")

# Find all Pokmon with a Height of 5 or more
tall_pokemon = df[df["Height"] >= 5]

print(f"Total entries: {len(tall_pokemon)}")
print(tall_pokemon[["Name", "Height"]].head())
```

---

## 2. Multiple Conditions

Combine multiple conditions using bitwise operators:
- `&` (**AND**): Both conditions must be true.
- `|` (**OR**): Either condition can be true.

> [!IMPORTANT]
> When using multiple conditions, you **must** wrap each individual condition in parentheses `()`.

```python
# Height >= 5 AND Weight >= 300
heavy_and_tall = df[(df["Height"] >= 5) & (df["Weight"] >= 300)]
print(heavy_and_tall[["Name", "Height", "Weight"]])

# Height >= 5 OR Legendary status
special_cases = df[(df["Height"] >= 5) | (df["Legendary"] == 1)]
```

---

## 3. Filtering by List (`.isin()`)

If you want to check against a list of specific values, `.isin()` is much cleaner than multiple OR statements.

```python
# Select Pokmon with Type1 as Fire, Water, or Grass (The Starters)
starter_types = ["Fire", "Water", "Grass"]
starters = df[df["Type1"].isin(starter_types)]

print(starters["Type1"].value_counts())
```

---

## 4. String Filtering

You can search for specific text patterns or substrings within a column.

```python
# Find Pokmon whose names contain "Mega"
mega_pokemon = df[df["Name"].str.contains("Mega")]
print(mega_pokemon["Name"].head())
```

---

## 5. Filtering Null Values

Identify missing data using `.isnull()` and non-missing data using `.notnull()`.

```python
# Rows where 'Type2' is missing (NaN)
missing_type2 = df[df["Type2"].isnull()]

# Rows where 'Type2' is NOT missing
known_type2 = df[df["Type2"].notnull()]
```
