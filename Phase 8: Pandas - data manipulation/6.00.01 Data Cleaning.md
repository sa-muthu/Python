# Data Cleaning in Pandas 

Data cleaning is the process of preparing your data for analysis by removing or modifying data that is incorrect, incomplete, irrelevant, or duplicated. Accurate insights start with clean data.

---

## 1. Dropping Irrelevant Columns

Some columns (like unique IDs or internal codes) aren't useful for statistical analysis. You can remove them using `.drop()`.

```python
import pandas as pd

# Load the dataset
df = pd.read_csv("Pokemon.csv")

# Remove the 'No' column
# axis=1 refers to columns. inplace=False returns a new DataFrame.
df_reduced = df.drop(columns=["No"])

print(f"Original Columns: {len(df.columns)}")
print(f"Cleaned Columns:  {len(df_reduced.columns)}")
```

---

## 2. Handling Missing Data (NaN)

Missing values (often represented as `NaN` or `None`) can break your calculations. You have two primary options:

### Option A: Remove Rows with Missing Values
Use `.dropna()` to remove rows that are incomplete.

```python
# Remove any row where 'Type2' is missing
df_dropped = df_reduced.dropna(subset=["Type2"])
print(f"Rows after dropping NaNs: {len(df_dropped)}")
```

### Option B: Fill Missing Values
Use `.fillna()` to replace missing values with a sensible default.

```python
# Replace missing 'Type2' values with the string "NONE"
df_filled = df_reduced.fillna({"Type2": "NONE"})
print(df_filled["Type2"].value_counts().head())
```

---

## 3. Standardizing Values

Inconsistent labels (e.g., "Grass" vs "grass") can cause grouping errors. Use `.replace()` or string methods to fix this.

```python
# Standardize 'Type1' to be all UPPERCASE
df_filled["Type1"] = df_filled["Type1"].str.upper()

# Fix a specific typo or naming convention
df_filled["Type1"] = df_filled["Type1"].replace({"GRASS": "PLANT"})
```

---

## 4. Converting Data Types

Sometimes data is loaded as a string when it should be a number (or vice versa). Use `.astype()` to correct this.

```python
# Cast 'Legendary' into a Boolean (True/False)
df_filled["Legendary"] = df_filled["Legendary"].astype(bool)

print(df_filled["Legendary"].dtype)
```

---

## 5. Removing Duplicates

Duplicate rows can skew your results. Use `.drop_duplicates()` to ensure each record is unique.

```python
# Identify and remove exact duplicates
initial_count = len(df_filled)
df_final = df_filled.drop_duplicates()

print(f"Removed {initial_count - len(df_final)} duplicate rows.")
```

---

##  Pro Tip: Chaining
You can perform multiple cleaning steps in one "chain" of commands for cleaner code.

```python
df_cleaned = (df.drop(columns=["No"])
              .fillna({"Type2": "None"})
              .drop_duplicates())
```
