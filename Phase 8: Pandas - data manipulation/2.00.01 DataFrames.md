# Pandas DataFrames Data

A **DataFrame** is a two-dimensional, size-mutable, and potentially heterogeneous tabular data structure. Think of it as a table in SQL or an Excel spreadsheet.

---

## 1. Creating a DataFrame

The most common ways to create a DataFrame are from dictionaries or external files (like CSVs).

### From a Dictionary
Each key in the dictionary becomes a **column**, and the values (lists) become the **rows**.

```python
import pandas as pd

data = {
    "Name": ["Pikachu", "Bulbasaur", "Charmander"],
    "Type": ["Electric", "Grass", "Fire"],
    "HP": [35, 45, 39]
}

# Note: 'D' and 'F' are capitalized
df = pd.DataFrame(data)
print(df)
```

### Custom Indexing
By default, rows are indexed from 0. You can specify a custom index labels.

```python
df_custom = pd.DataFrame(data, index=["P01", "P02", "P03"])
print(df_custom)
```

---

## 2. Importing Data from Files

In real projects, you'll rarely type data manually. You'll import it from local or remote files.

```python
# Load from a CSV file
# Ensure the CSV is in the same folder as your script
df_pokemon = pd.read_csv("Pokemon.csv")

# Quick Look: View the first 5 rows
print(df_pokemon.head())

# Technical Look: View column types and missing values
print(df_pokemon.info())
```

---

## 3. Basic DataFrame Operations

### Row & Column Selection
- **`.loc[]`**: Access by **Label**.
- **`.iloc[]`**: Access by **Integer Position**.

```python
# Select second row (index 1)
print(df_pokemon.iloc[1])

# Select one specific cell
print(df_pokemon.loc[0, "Name"])
```

### Adding & Removing Data
```python
# Adding a new column
df_pokemon["Note"] = "Review Pending"

# Removing a column (axis=1)
# inplace=True modifies the original DataFrame
df_pokemon.drop(columns=["Note"], inplace=True)
```

---

## 4. Exporting Data

After cleaning or modifying your data, you can save it back to a file.

```python
# Save to a new CSV file
# index=False prevents Pandas from adding a row-number column
df_pokemon.to_csv("Cleaned_Pokemon.csv", index=False)
```
