# Pandas DataFrames

A DataFrame is a two-dimensional, size-mutable, and potentially heterogeneous tabular data structure with labeled axes (rows and columns). Think of it like a spreadsheet or a SQL table.

## 1. Creating a DataFrame from a Dictionary

You can create a DataFrame by passing a dictionary where keys are column names and values are lists of data.

> [!NOTE]
> The 'D' and 'F' in **DataFrame** are capitalized: `pd.DataFrame()`.

```python
import pandas as pd

data = {
    "Name": ["Romeo Montague", "Harry Potter", "Frodo Baggins"],
    "Age": [17, 17, 50]
}

novel_characters = pd.DataFrame(data)
print(novel_characters)
```

### Changing the Index
By default, rows are indexed with integers starting from 0. You can specify custom labels using the `index` argument.

```python
novel_characters_w_index = pd.DataFrame(data, index=["Character 1", "Character 2", "Character 3"])
print(novel_characters_w_index)
```

## 2. Accessing Data

You can access rows using their labels or integer positions.

- **`.loc["label"]`**: Access by index label.
- **`.iloc[index]`**: Access by integer position.

```python
print(novel_characters_w_index.loc["Character 2"]) # Using the indexed name
print(novel_characters_w_index.iloc[1])             # Using the index position
```

## 3. Modifying Data

### Changing a Value
Use `.loc` or `.iloc` with the assignment operator to update specific rows or cells.

```python
novel_characters_w_index.loc["Character 2"] = ["Gerold Weasley", 17]
print(novel_characters_w_index)
```

### Adding a New Column
Adding a column is as simple as assigning a list to a new column name.

```python
novel_characters_w_index["Novel/Play"] = ["Romeo and Juliet", "Harry Potter", "Lord of the Rings"]
print(novel_characters_w_index)
```

## 4. Adding Rows

### Adding a Single Row
To add a new row, create a temporary DataFrame and use `pd.concat()`.

```python
new_row = pd.DataFrame([{"Name": "Jon Snow", "Age": 23, "Novel/Play": "Game of Thrones"}], index=["Character 4"])
novel_characters_w_index = pd.concat([novel_characters_w_index, new_row])
print(novel_characters_w_index)
```

### Adding Multiple Rows
You can concatenate multiple rows at once by providing a list of dictionaries.

```python
new_rows = pd.DataFrame([
    {"Name": "Alan Grant", "Age": 40, "Novel/Play": "Jurassic Park"},
    {"Name": "Odysseus", "Age": 40, "Novel/Play": "The Odyssey"}
])
novel_characters_w_index = pd.concat([novel_characters_w_index, new_rows])
print(novel_characters_w_index)
```

## 5. Removing Rows and Columns

To remove data, use the `.drop()` method.

- **`axis=0`**: Refers to rows.
- **`axis=1`**: Refers to columns.
- **`inplace=True`**: Modifies the original DataFrame directly.

### Removing Rows
```python
# Remove a single row
novel_characters_w_index.drop("Character 4", axis=0, inplace=True)

# Remove multiple rows
novel_characters_w_index.drop(["Character 1", "Character 2"], axis=0, inplace=True)
print(novel_characters_w_index)
```

### Removing Columns
```python
# Remove a single column
novel_characters_w_index.drop("Novel/Play", axis=1, inplace=True)

# Remove multiple columns
novel_characters_w_index.drop(["Name", "Age"], axis=1, inplace=True)
print(novel_characters_w_index)
```
