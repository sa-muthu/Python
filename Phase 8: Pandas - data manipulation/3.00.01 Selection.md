# Selection in Pandas 🎯

Selecting data is the foundation of analysis. Pandas offers multiple ways to grab specific columns, rows, or subsets of your table.

---

## 1. Selecting Columns

To select a single column (returning a Series), use bracket notation with the column name.

```python
import pandas as pd

df = pd.read_csv("Pokemon.csv")

# Select one column
names = df["Name"]
print(names.head())

# Select multiple columns (pass a LIST of names)
subset = df[["Name", "Type1", "HP"]]
print(subset.head())
```

---

## 2. Selecting Rows by Label (`.loc`)

Use `.loc` when you want to select rows based on their **index label**.

```python
# Select the row with index 0
first_pokemon = df.loc[0]
print(first_pokemon)

# Select a range of rows (inclusive)
top_ten = df.loc[0:9]
print(top_ten)
```

---

## 3. Selecting Rows by Position (`.iloc`)

Use `.iloc` when you want to select rows based on their **integer position** (starting from 0).

```python
# Select the first row
print(df.iloc[0])

# Select the last row
print(df.iloc[-1])

# Slicing: Rows 5 to 10 (exclusive of 10)
print(df.iloc[5:10])
```

---

## 4. Selecting Specific Rows AND Columns

Both methods allow you to specify columns as the second argument.

```python
# .loc[row_label, col_label]
print(df.loc[0, "Name"])

# .iloc[row_pos, col_pos]
print(df.iloc[0, 1]) # Row 0, Column 1
```
