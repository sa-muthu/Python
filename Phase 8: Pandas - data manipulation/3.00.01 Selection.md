# Selection in Pandas

Selecting specific data is one of the most common tasks in data analysis. Pandas provides several ways to select columns, rows, or specific cells.

## Loading Data

Before selecting data, we first need to load a dataset. In this example, we use a CSV file containing Pokémon data.

```python
import pandas as pd

data = pd.read_csv("Pokemon.csv")
print(data)
```

---

## 1. Selection by Column

To select a column, use the column name inside square brackets `[]`.

### Select a Single Column
```python
# Returns a Series
print(data["Name"])

# To print the whole column (as a string)
print(data["Name"].to_string())
```

### Select Multiple Columns
To select multiple columns, pass a **list** of column names inside the brackets.
```python
# Returns a DataFrame
print(data[["Name", "Height", "Weight"]])
```

---

## 2. Selection by Row

Use the `.loc[]` attribute to select rows by their index label.

| Command | Description |
| :--- | :--- |
| `data.loc[0]` | Selects a single row (label 0). |
| `data.loc[0:10]` | Selects a range of rows (from 0 to 10 inclusive). |
| `data.loc[0:10:2]`| Selects rows with a step (every 2nd row). |

### Select Rows and Columns Simultaneously
You can specify both the rows and the columns you want to retrieve.
```python
# Select rows 0 to 10 for specific columns
print(data.loc[0:10, ["Name", "Height", "Weight"]])
```

---

## 3. Selection by Column and Row (Specific Cell)

To retrieve a single value from a specific row and column:

```python
# Selects the value in row 0, column "Name"
print(data.loc[0, "Name"])
```
