# Setting Up Your Python Environment

Welcome to your Python setup guide! Follow these steps to get your environment ready for Python development.

## 1. Install Python

To get started, you need to download and install Python on your machine.

- Go to the official Python website: [https://www.python.org/downloads/](https://www.python.org/downloads/)
- Click on the download button (e.g., **Download Python 3.14.3** or the latest version).
- Run the installer.

## 2. Verify Your Installation

Once the installation is complete, verify that Python was installed correctly.

Open your terminal and type the following command:
```bash
python --version
```
*(If the command `python` doesn't work, try running `python3 --version` instead).*

You should see the installed Python version printed in the terminal.

## 3. Install an IDE (Integrated Development Environment)

An IDE is a software application that provides everything you need to write and run code in one place. Key features typically include:

- **Code editor**
- **Execution/run environment**
- **Debugging tools**
- **Syntax highlighting**
- **Extensions and plugins**

### Recommended IDEs

Here are two popular options for Python development:

#### **Visual Studio Code (VS Code)**
A lightweight, highly customizable code editor that becomes a powerful IDE with extensions.

- **Download:** [https://code.visualstudio.com/](https://code.visualstudio.com/)
- **Setup:** After installing, open VS Code and go to the **Extensions** tab on the left sidebar. Search for **Python** (published by Microsoft) and install it.

#### **PyCharm**
A full-fledged IDE that comes out of the box with rich built-in features specifically tailored for Python.

- **Download:** Search for PyCharm (the Community edition is free) and install it. It provides a more comprehensive out-of-the-box experience compared to VS Code.
